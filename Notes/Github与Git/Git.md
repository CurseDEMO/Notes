- 版本控制系统
- 以**commit**为控制单元(快照)
- 分为**本地仓库**和**远端仓库**(GitHub)
每一次更改暂存提交组成一个快照,这些快照可恢复可访问,先学会用图形化的Visual Studio Code里的Git功能完成一次简单的仓库推送
# 安装
1. 先安装VSCode,勾选所有能勾上的勾,==无脑==下一步
2. 打开Git安装程序,无脑下一步,注意在选择编辑器界面选择VSCode(下拉菜单选择,==必须先安装VSCode==,否则无法安装Git)之后无脑下一步.
